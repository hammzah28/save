Hardware & Software

Complete Computer Science Notes

1. What is Hardware?

Hardware refers to the physical components of a computer system. These are the parts that can be physically touched.

Examples of Hardware

  • Keyboard
  • Mouse
  • Monitor
  • CPU (Central Processing Unit)
  • Hard Disk / SSD
  • Printer

2. Categories of Hardware

a) Input Devices

Input devices are used to enter data into the computer.

  • Keyboard – enters text and numbers
  • Mouse – controls the pointer
  • Scanner – converts paper documents into digital form
  • Microphone – inputs sound
  • Webcam – inputs video

b) Output Devices

Output devices show the results of processing.

  • Monitor – displays output
  • Printer – prints hard copies
  • Speakers – output sound
  • Projector – displays visuals on large screens

c) Processing Devices

Processing devices perform calculations and decision making.

  • CPU (Central Processing Unit)
  • GPU (Graphics Processing Unit)

d) Storage Devices

Storage devices store data permanently or temporarily.

  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • USB Flash Drive
  • Memory Card
  • CD / DVD

3. Primary and Secondary Storage

Primary Storage

Primary storage is directly accessible by the CPU.

  • RAM (Random Access Memory) – volatile
  • ROM (Read Only Memory) – non‑volatile

Secondary Storage

Secondary storage stores data permanently.

  • HDD
  • SSD
  • Optical Disks
  • Cloud Storage

4. What is Software?

Software is a set of programs or instructions that tell the computer hardware what to do.

Without software, hardware is useless.

5. Types of Software

a) System Software

System software manages computer hardware and provides a platform for running applications.

Examples

  • Operating Systems
  • Device Drivers
  • Utility Programs

Operating Systems

  • Windows
  • Linux
  • macOS
  • Android
  • iOS

b) Application Software

Application software is designed to help users perform specific tasks.

  • Word processors (MS Word)
  • Spreadsheets (Excel)
  • Presentation software (PowerPoint)
  • Web browsers (Chrome, Firefox)
  • Games

6. Utility Software

Utility software helps to maintain and optimize the computer system.

  • Antivirus software
  • Disk cleanup tools
  • Backup software
  • File compression tools

7. Firmware

Firmware is a type of software stored in ROM. It controls how hardware behaves.

Example: BIOS

8. Relationship Between Hardware and Software

  • Hardware performs tasks
  • Software gives instructions
  • Both depend on each other

Example: A printer (hardware) needs drivers (software) to work.

9. Advantages and Disadvantages

Hardware

  • ✔ Fast processing
  • ✘ Expensive to replace

Software

  • ✔ Easy to update
  • ✘ Can contain bugs

10. Exam Tips

  • Always define hardware and software clearly
  • Give examples in answers
  • Use headings in long questions
  • Compare hardware vs software if asked
